## DP(dynamic programming)
把複雜問題拆解成相對簡單的子問題<br>
再利用保存了解決這些子問題的結果來減少複雜度<br>
### 基本要件 
最優子問題<br>
- 分而治之
- 遞迴
<br>
重複子問題<br>

### Fib數列
brute force O(2^n) <br>
DP O(n) <br>

### 從revursion到DP
- 自下而上
  遞迴+記憶中間結果<br>
  O(n) space
  O(n) time
- 自上而下
  從最後一步遞迴結果往前推
  O(1) sapce
  O(n) sapce
